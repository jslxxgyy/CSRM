好，现在需要你写出一个Python程序，满足以下要求，1:将配置文件的[RadioNetwork]节中的内容转化为如下json格式:
{
"name":"电台网络名",
"nameId": "电台网络名",
"description": "这是一个示例电台网络",
"descriptionId": "这是一个示例电台网络",
"icon": "coui://extendedradio/CustomRadios/电台网络/电台网络图标.svg",
"allowAds": true
}
将结果输出至"./temp/RadioNetwork.json"文件中。
2.将配置文件中的[RadioChannel]节中的内容转化为以下json格式:
{
"name": "频道1",
"nameId": "频道1",
"description": "这是电台网络的频道1",
"icon": "coui://extendedradio/CustomRadios/电台网络/频道1/频道图标.svg
}
将结果输出至"./temp/RadioChannel.json"文件中。
3.将配置文件中的[Program]节中的内容转化为以下json格式:
{
    "name": "节目1",
    "description": "这是电台网络的频道1的其中一个节目",
    "startTime": "00:00",
    "endTime": "00:00",
    "loopProgram": true,
    "pairIntroOutro": false
}
4.将配置文件中的[Segment]节中的内容转化为以下json格式，注意，这个部分你只需要将[Segment]节中的clipsCap项对应到下面json示例中的"clipsCap"的内容就行了，不要更改其他内容。
{
    "type": "Playlist",
    "tags": [
        "type:Music",
        "radio channel:音乐"
    ],
    "clipsCap": 3
}
